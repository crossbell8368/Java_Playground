
1. 컴퓨터의 수가 100 이하이므로, 주어진 메모리 제한은 인접 행렬과 인접 리스트 두 방식 무엇으로 표현해도 넉넉하다. (O)
    => 컴퓨터는 Node를 의미
    => 100 이하의 갯수면 인접행렬로 구현해도 무리가 없음

2. 구하려는 정답은 1번 컴퓨터가 속한 연결 요소의 크기와 같다. (O)
    => 1번 컴퓨터가 속한 연결요소의 정점의 갯수와 일치한다.

3. 어떤 컴퓨터도 1번 컴퓨터와 연결되어 있지 않거나, 주어진 연결쌍의 갯수가 0이라면 출력해야하는 정답은 0이다. (O)
    => 1번 컴퓨터는 세지 않으므로 정답은 0이다.

4. 모든 컴퓨터가 바이러스의 걸렸다면, 주어진 연결쌍의 갯수는 N-1 개 이상이다. (O)
    => 모든 컴퓨터가 1번 컴퓨터가 소속된 그룹에 소속
    => 이때 정점을 연결하는 최소 간선의 갯수는 N-1 개
