
1. 연결 요소란 그래프 내에서 서로 연결된 정점들의 집합으로, A 연결 요소에 포함된 정점은 다른 연결 요소에 포함된 정점과 연결되지 않는다. (O)
    * 다른 연결요소에 연결 => 이는 하나의 연결요소를 의미
    * 그렇기에 다른 연결요소와 연결되지 않는다.

2. 1번 정점에서 방문할 수 있는 모든 정점은 1번과 동일한 연결요소에 속하며, 해당 정점들에서도 1번 정점을 방문할 수 있다. (O)
    * 같은 연결 요소에 속하는 것이기에 당연한 이야기

3. 정답으로 출력 가능한 연결요소의 최대 갯수는 N-1 개이다. (X)
    * 문제요건에서 간선의 최소값이 0이다.
    * 즉 정점들만 부여될 수 도 있다는 것이다.
    * 정점들만 존재할 경우, 연결요소는 N개이다.

4. 모든 정점이 하나의 연결 요소에 포함되기 위해 필요한 간선의 최소 개수는 N개이다. (X)
    * 모든 정점이 하나의 연결요소에 포함되는 최소의 간선 갯수는 N-1 개이다.
